 【明慧网二零一九年七月十八日】为了增進世人对法轮功（也叫
 <ok href="http://www.minghui.org/mh/glossary.html#1">
  法轮大法
 </ok>
 ）的了解，明慧编辑部新成立了明慧出版中心。这个非盈利机构的宗旨是发行印刷版和电子版的英文书籍、杂志、文章和多媒体材料，以及从事相关的活动。这些书籍和杂志的主题集中在三方面：1）修炼法轮大法的益处，包括身体上和精神上的健康改进；2）第一手关于对法轮功的迫害事实；3）关于自身修养的传统文化。

  <p>
   明慧出版中心已开始发表明慧人权报告英文版。此报告全面概括了法轮功学员为了坚持信仰而承受的来自中共的迫害，来自于中文
   <ok href="http://www.minghui.org/mh/glossary.html#2">
    明慧网
   </ok>
   二十年来收集的数据和一手信息。
   <br/>
   <br/>
   英文报告整理为六部：
   <br/>
   第一部　迫害报告（特点，手段，程度，人权侵犯）
   <br/>
   第二部　法轮功现状
   <br/>
   第三部　推动迫害的主要人员与组织
   <br/>
   第四部　二十一万人诉江
   <br/>
   第五部　法轮功基本真相（肃清中共对法轮功的污蔑宣传）
   <br/>
   第六部　前所未有的罪恶：活摘器官
   <br/>
   <br/>
   我们希望此报告会帮助更多人了解中共政权对法轮功的迫害与人权侵犯。
