     <p>
      PDF 文件：
      <br/>
      第一步：先打印单页，在“文件---&gt;打印”设置页面范围，设置单页，再按“确定”按钮即打印；
      <br/>
      第二步：将纸翻面；
      <br/>
      第三步：再打印双页，在“文件---&gt;打印”设置页面范围，设置双页，再按“确定”按钮即打印。
      <br/>
      第四步：从长边中间裁切开，沿中线装订，即为两份小册子。
      <p>
       WORD 文件：
       <br/>
       第一步：在“文件---&gt;打印”里设置页面范围，输入下列页数：
       <br/>
       20,1,20,1,18,3,18,3,16,5,16,5,14,7,14,7,12,9,12,9
       <br/>
       设置缩放选项为每页4版，按纸型缩放为A4纸。再按“确定”按钮即打印；
       <br/>
       第二步：将纸翻面；
       <br/>
       第三步：在“文件---&gt;打印”里设置页面范围，输入下列页数：
       <br/>
       2,19,2,19,4,17,4,17,6,15,6,15,8,13,8,13,10,11,10,11
       <br/>
       设置缩放选项为每页4版，按纸型缩放为A4纸。再按“确定”按钮即打印，
       <br/>
       第四步：从长边中间裁切开，沿中线装订，即为两份小册子。
