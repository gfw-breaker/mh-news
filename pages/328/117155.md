 【明慧网2005年12月23日】2005年12月11日晚，位于美国首都华盛顿DC的美利坚大学的艺术中心正在举办开放给公众的独幕剧联演，每一个剧的导演都是导演班的学生。在舞台灯光的照射下，一幕一幕的剧连续登场。
 <p>
  下半场的第一幕一开始，一曲平静宽广柔和的英文歌曲《为你而来》在场中响起，那悠扬深沉的由
  <ok href="http://www.minghui.org/mh/glossary.html#1">
   法轮功
  </ok>
  学员们创作演唱的歌曲让观众们的心境突然平静下来。这时，一位西方法轮功学员在天安门背景前走出，开始了《天安门偶遇》的独幕剧。
  <p>
   地点：北京天安门广场。人流如梭。
   <br/>
   人物：一对年轻夫妇和一名西人法轮功学员
   <p>
    西人法轮功学员（舞台外）：此次去北京，除了万里长城之外，我还要拍摄更多其它的中国风光。所以现在我来到了天安门广场。
    <p>
     （年轻夫妇上）
     <br/>
     男：喂，你好！
     <br/>
     西人学员（眼睛慢慢的从摄像机屏幕处移开）：噢，你好。
     <p>
      女：你在拍什么？
      <br/>
      西人学员：我在拍天安门广场。
      <p>
       男：哇，你看起来很专业？
       <br/>
       女：是的，非常专业。
       <br/>
       西人学员（沮丧的将手放到前额处）：我给你们看一些我拍的照片。糟糕！电池没电了。唉，没办法了。喂，那你们在北京干什么呢？
       <p>
        男：我们是艺术系的学生。学绘画的。
        <br/>
        西人学员：我也是。
        <p>
         男、女（同声）：你也学绘画！
         <br/>
         西人学员（微笑）：我们都是“艺术家”。
         <p>
          女：你画什么题材？
          <br/>
          西人学员：我喜欢静物写生。
          <p>
           男：你画图片吗？
           <br/>
           女：那你有没有兴趣看中国画？
           <br/>
           西人学员：你是说中国传统绘画吗？是的，我喜欢中国艺术，在哪儿可以见到呢？
           <br/>
           男：附近有一家画廊，里面陈列了很多作品，有一些是我画的。来，我带你去看看。
           <p>
            西人学员：你怎么英语说这么好？
            <br/>
            男：噢，是吗？不，我说得不好。
            <br/>
            女：我们的英语老师是从美国来的。我们跟他学。你的中文也不错！
            <br/>
            西人学员：噢，是吗？不，我的汉语还不行。
            <p>
             男：我们到了。
             <br/>
             女：看看这些画。你喜欢哪幅？这张，还是那张？
             <br/>
             西人学员：让我先仔细的把所有的画都欣赏过再说。
             <br/>
             男：这张竹代表真理。你知道，竹子非常坚韧、挺拔，就像事实一样。
             <br/>
             女：竹子的中心是空的，但它却很有力、很纯洁，没有东西能改变它。你喜欢这幅画吗？
             <p>
              （画外音：这几年，中国大陆的法轮功学员，无论是在天安门广场，还是在城市、家庭、学校和工作单位 - 在任何场合，他们都坚持不懈的讲法轮功的真相。尽管可能面临拘捕、酷刑、殴打、强奸、勒索、恐吓或开除公职，他们都不停的站出来，而不是胆怯的躲藏起来。并且这些法轮功学员，都不标榜他们自己或个人，而是将其巨大的勇气和力量归功于法轮大法。你不觉得他们也象这竹子一样吗？）
              <p>
               女：请看这幅。这上面的荷花象征高贵的品质和慈悲的胸怀。
               <br/>
               西人学员：是的，非常好。
               <br/>
               男：嗯，那幅怎样？
               <br/>
               西人学员：中国书法。这幅写的是“忍”字。啊，“忍，”写得真好！我知道这个“忍”字代表节制、耐心、宽容和忍受痛苦。
               <p>
                男：是的。它的内涵是很深的，意义也是很深远的。
                <br/>
                西人学员：这我知道。
                <br/>
                男：嗯，现代的人都不会欣赏传统艺术了，真是令人悲哀。顺便问一下，你为什么对中国文化这么感兴趣？
                <p>
                 西人学员：噢，这是因为我炼法轮功的缘故。
                 <br/>
                 女：什么？你炼法轮功？
                 <br/>
                 男（对女）：小声点。
                 <br/>
                 女：你为什么要炼法轮功？
                 <br/>
                 西人学员：我修炼法轮功已有五年了。法轮功是一项古老的有益身心的功法，基于真、善、忍的原则。非常好。
                 <p>
                  女：那你如何解释那些在天安门广场上自焚的人呢？
                  <br/>
                  西人学员：我很高兴你问到这个问题。那是一场骗局。那些人根本不是法轮功学员。你看过中共当局制作的节目吧？
                  <p>
                   女（点头）：是的，我看过。
                   <br/>
                   西人学员：当我们拿到节目的带子后，进行了慢镜头分析，你猜我们看到了什么？
                   <br/>
                   女（摇摇头）：你们看到什么了？
                   <p>
                    西人学员：我们看到了可怕的真相。当警察用灭火器去扑灭围绕其中一名妇女的火焰时，她被浓烟包围着。你可看到此时一名警官用一重物去击打这名妇女的头部，她便立即倒在了地上。当我们将这镜头分析公布于世时，你们的政府收回了这个已播放的版本并删掉了该镜头。而且，在一周以内的新闻报道中，自焚者数目被改动多次。从第二周起，新闻开始报道一个小女孩也在自焚者之列。事件发生后，媒体怎么可能在第一周就忘了报道一名小女孩呢？而且那么多警察如何那么快就拿到了灭火器呢？外国记者被禁止采访，当他们要报道自焚事件或法轮功时，却受到迫害。你们不知道这些是因为一旦政府内有人要披露事实真相时，便会遭到迫害。一些政府高官极尽所能的帮前国家主席江××掩盖。事实上，没人愿相信中共应对这起自焚案负责，因为太惨了，人们简直不敢相信政府会做出这样的事情来。待到真相大白的一天，许多人会因轻信了谎言而羞愧得无地自容。这真是一场悲剧！中国的悲剧。
                    <p>
                     西人学员：你们的政府 - 就是同样迫害法轮功的那个 - 曾估计有七千万人炼法轮功。有这么多人炼，却只有5人害自己，你怎么能责怪法轮功呢？这就为什么说自焚事件是明显的栽赃宣传，是媒体专门用来攻击法轮功的。
                     <br/>
                     （警车声）
                     <br/>
                     男：小心点，我们回到天安门广场了。
                     <br/>
                     西人学员：好，我再来谈谈我自己。你们知道吗？炼法轮功前，我长期身体虚弱，对生活没有信心。但是，我开始炼法轮功后便变了，我从此感到健康，快乐和强壮。看到我的变化后，我的母亲和我的一些朋友也开始修炼法轮功。
                     <p>
                      女（面带深深的敬意）：是真的吗？
                      <br/>
                      西人学员：你们知道如今在全世界有六十多个国家里都有人炼法轮功吗？
                      <br/>
                      男、女：不，我们不知道。
                      <p>
                       西人学员：你们知道法轮大法及其创始人
                       <ok href="http://www.minghui.org/mh/glossary.html#32">
                        李洪志
                       </ok>
                       先生收到过几百份褒奖以表彰其对社会的贡献吗？
                       <br/>
                       男、女：不，我们不知道。
                       <p>
                        西人学员：你们知道李洪志先生连续两年被提名授予诺贝尔和平奖吗？
                        <br/>
                        男、女：不，我们不知道。
                        <p>
                         西人学员：如果你们能出国到中国以外的任何一个国家，你可从网上找到有关法轮功的任何信息。但你如果在中国查找的话，你一旦键入“法轮功”三个字，你的IP地址就会被跟踪，你甚至可能会因此被捕入狱。
                         <br/>
                         女：（对男）看来法轮功不象政府讲的那样。
                         <br/>
                         （对西人学员）我想知道更多有关法轮功的消息。
                         <p>
                          西人学员：太好了。你也可以即刻将我告诉你的讲给你所有的亲朋好友。
                          <p>
                           （男的听到警车声，变得害怕起来）
                           <p>
                            男：不要那么说。这是被禁止的。
                            <br/>
                            西人学员（很惊讶）：对不起，你说什么？
                            <br/>
                            男：这是反政府的言论，是被禁止的。
                            <p>
                             西人学员：你错了。这是教人如何做一个好人，哪里反政府了？法轮功不反对政府，而且永远不会。我们只是修炼。
                             <br/>
                             男：我看见炼法轮功的在天安门广场上举起横幅并高喊“法轮大法好”。他们干嘛要这样向政府抗议？
                             <br/>
                             西人学员：我们还有别的办法吗？近3000名学员被酷刑折磨致死，无数人被打，被酷刑折磨和注射药物，就因为他们炼法轮功。你如果没有别的办法鸣冤，你会怎么办？这就像你刚才指给我看的那幅竹一样。我们必须捍卫真理，即使面对迫害。中国人民应该知道法轮功是好的。
                             <p>
                              （汽车声）
                              <br/>
                              男：车来了。
                              <br/>
                              女：谢谢你告诉我们这些。
                              <br/>
                              西人学员：这是你们应该知道的。人们应该了解有关法轮功的真相。
                              <p>
                               男、女向公共汽车方向走去。
                               <br/>
                               西人学员演示法轮功功法。
